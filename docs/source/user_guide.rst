User Guide
==========

This is the user guide for SHRED-X. It will cover the basics of using the package, including how to install it, how to use the API, and how to contribute to the package.

Installation
------------

The package can be installed by cloning the repository and installing the environment::

    git clone https://github.com/yyexela/SHRED-X.git
    cd SHRED-X

    pyenv install 3.13.7
    pyenv local 3.13.7
    python -m venv ~/.virtualenvs/shredx

    source ~/.virtualenvs/shredx/bin/activate
    pip install -e .
